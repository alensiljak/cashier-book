# Expenses Report

The Expenses report displays all Expenses incurred in the selected timeframe.

## Time Periods

Standard time periods are selectable in the dropdown list. This narrows the expenses to the selected period.

## Filtering

By clicking the Filter button in the toolbar, the user is able to select Expense accounts they do not want to see in the report.

Next to the filter is a quick toggle for disabling and enabling the filter. Disabling the filter displays all the Expense categories in the report.

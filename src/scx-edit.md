# Scheduled Transactions, Edit

The Scheduled Transaction editor consists of two sections:

1. transaction editor
2. schedule editor

## Transaction Editor

This is a standard transaction editor. The transaction created here is a template that will be used to create Journal transactions from the Scheduled Transaction.

## Schedule Editor

The schedule editor provides option to set up recurrence of the transaction.

The Repayment option signals that this is a repayment transaction, i.e. credit card repayment or a loan repayment, and the amount will automatically be calculated by the app by taking the full amount of the Liabilities account in the transaction.
The second account should be an Asset, which will be credited.

# Introduction

This is user documentation for the [Cashier](https://cashier.alensiljak.cc/) application.

Cashier is a part of the Plain-Text Accounting (PTA) ecosystem.

It uses Beancount file format and is compatible with Beancount and Rust Ledger.

## History

[Cashier I](https://cashier.alensiljak.eu.org) worked with Ledger.
Ledger (ledger-cli) is a command-line program for personal finance. It runs on multiple plaftorms.

# Set-up

## Quick Setup

To start using the application, you need to perform an initial setup.

1. Import your ledger files.
2. Set customization options.

These are performed in the [Settings](settings.md) screen.

## Data Source

Configure the data source by going to [Sync](sync.md) and selecting the desired backend.
Once selected, click the `Configure` button to perform the backend configuration.

- [File System Synchronization](file-system-sync.md)

## Customization

Customization is done in the [Settings](settings.md) screen.

## Initial Synchronization

While on the [Sync](sync.md) screen, perform the initial synchronization with all the options selected.
